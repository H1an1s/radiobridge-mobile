# RadioBridge Mobile

Статический интерфейс GitHub Pages для команд из Bark через ntfy.

1. Замените `CHANGE_ME_TO_A_LONG_RANDOM_TOPIC` в `config.js` на тот же длинный случайный topic, который настроен в ESP32.
2. Опубликуйте каталог как отдельный GitHub-репозиторий с GitHub Pages.
3. В административной панели RadioBridge задайте URL Pages в поле «URL мобильного интерфейса».

Topic не является средством авторизации. Backend принимает действие только по одноразовому токену, привязанному к пользователю, устройству, вызову и версии вызова.
